Pothole Patrol

Pothole patrol is a Artificially Intelligent Pothole Detection, Reporting and Management Solution. It is available on the Web and contains a Responsive Suite of Tools for Reporting and Management..

  • AI powered validation

    Deep learning model that validates each upload before allowing to create a report. Thus, reducing irrelevant data and ensuring efficient processing on management end.

  • Citizen's Reporting Application

    A reporting system for the citizens where they can capture the scene of an area, which is geocoded and fed to the core AI to create a validated report.

  • Authority's Management Application

    An application for authorities to administer reports using different viewing techniques to manage work orders efficiently for their jurisdictions.

  • Completely Responsive Feature Suite

    Desktop and mobile compatible citizen and authority applications. Supports a number of features and can be used with any of the popular browsers.
